# Shorten Url Api

## Prerequisites

- [NodeJs](https://nodejs.org/en/)
- [MongoDB](https://www.mongodb.com/try/download/enterprise)

## Before running app

- Make sure you are running mongodb

## To run the app

```sh
$ git clone https://github.com/TshepzMogapi/sho-ten-api.git
$ cd sho-ten-api
$ npm install
$ npm run dev
```
